Introduction

This document explains how to communicate with the TaxPower API.

Authentication

api_key

To make authorized requests to the TaxPower API, you will need to signup for an account and generate an API key.

To signup for an account: https://www.taxpower.com/docs/getting-started/account-setup

To generate an API key: https://www.taxpower.com/docs/getting-started/generate-api-key

Once you have a valid API key, simply attach that key as the value for the "api_key" header to make authenticated requests.

Security Scheme Type API Key
Header parameter name: api_key

Tax

Calculate Tax

Calculate tax for any zipcode in the US. Multiple currencies supported.

Authorizations:
api_key
    query Parameters
    amount
    required
    number
    Example: amount=19.99

    This is the base amount that will be used to calculate the tax.

    zipcode
    required
    number
    Example: zipcode=95501

    The tax rate used to calculate the tax will be retrieved using this zipcode.

    currency
    string
    Default: "USD"
    Enum: "USD" "EUR"
    Example: currency=USD

    The desired currency you want the TaxPower API to respond with.

    header Parameters
    accept
    required
    string
    Default: application/json

    The response data will be in JSON format.

    api_key
    required
    string
    Example: apikey1234abcdefghij0123456789

    Used to authenticate the request.
